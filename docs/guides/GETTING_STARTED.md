# Getting Started

> **Status:** maintained | **Last reviewed:** 2026-06-29 | **Audience:** users

## Prerequisites

- Node.js 22+
- pnpm 9+
- Git

## Installation

```bash
git clone https://github.com/phamthanhhung/flowtask.git
cd flowtask
pnpm install
```

## First Run

```bash
# Initialize a project
pnpm dev init --name "My Project" --mode development

# Start a run
pnpm dev run "your prompt here"
```
